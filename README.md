# Geometrix - 3D Polyhedron Viewer

A modern, fast, client-side 3D viewer for exploring polyhedrons and 2D polygons.

## Features

- **3D Polyhedron Viewing**: Interactive 3D models of various polyhedrons
  - Tetrahedron
  - Cube
  - Octahedron
  - Dodecahedron
  - Icosahedron
  - Pyramid

- **2D Polygon Mode**: View regular polygons from 3-12 sides
- **Modern UI**: Glass morphism effects with Tailwind CSS
- **Interactive Controls**: 
  - Mouse rotation, zoom, and pan
  - Adjustable rotation speed
  - Wireframe mode toggle
  - Color customization
- **Keyboard Shortcuts**:
  - `Space`: Toggle auto-rotation
  - `R`: Reset view
  - `W`: Toggle wireframe
  - `3-9`: Create 2D polygons (in 2D mode)

## Usage

1. Open `index.html` in a modern web browser
2. Use the polyhedron selection buttons to switch between shapes
3. Toggle between 3D and 2D viewing modes
4. Adjust rotation speed, wireframe mode, and colors using the controls
5. Interact with the 3D models using mouse controls

## Technology Stack

- **Three.js**: 3D graphics rendering
- **Tailwind CSS**: Modern UI styling
- **Vanilla JavaScript**: No build process required

## Browser Compatibility

Works best in modern browsers that support WebGL:
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+
